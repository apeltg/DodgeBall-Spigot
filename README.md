### **DodgeBall - Spigot** #

DodgeBall è um plugin brasileiro sobre uma "pseudo queimada" de time, onde o jogador possui um arco e tem que eliminar o time inimigo inteiro, similar ao jogo finalista do [MCC](https://noxcrew.com/mcc)
